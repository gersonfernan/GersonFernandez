Primera practica de git Laboratorio II

Pasos:

  • Crear Repositorio: https://github.com/gersonfernan/GersonFernandez.git
  • Inicializar el Repositorio: git init
  • Creamos lab.html en el Repositorio Local
  • Agregar los Cambios para hacer subidos a git: git add -A
  • Realizar un Commit con los cambios agregados: git commit -m "first commit"
  • Marcar la rama de trabajo actual como principal: git branch -M main
  • Agregar el Repositorio remoto donde almacenar los cambios: git remote add origin https://github.com/gersonfernan/GersonFernandez.git
  • Subir los cambios del Repositorio local al Repositorio remoto: git push -u origin main
